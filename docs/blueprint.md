# **App Name**: ResumeForge

## Core Features:

- Fillable Fields: Provides fillable form fields corresponding to standard resume sections (e.g., Summary, Projects, Education, Experience, Skills) based on a pre-defined template for ATS compatibility.
- ATS Format: Format generated resume into the the most popular and effective layouts that is compatible with Applicant Tracking Systems.
- Suggested phrasing: LLM suggest phrasing for different sections based on your education, skills, experience and work duties to improve overall impact. The LLM acts as a writing tool.
- Download: Ability to download the generated resume in PDF format.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA), for a modern and calming feel.
- Background color: Light gray (#F5F5F5) to ensure readability and a clean, professional look.
- Accent color: Dusty rose (#D8BFD8) to add subtle sophistication without being distracting.
- Body and headline font: 'Inter', a sans-serif with a modern, objective, neutral look.
- Simple, professional icons for section headers (e.g., briefcase for experience, graduation cap for education).
- Clean, single-column layout for optimal ATS parsing. Use clear section dividers to enhance readability.
- Subtle fade-in effects when the resume is generated.
